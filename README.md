# Enrique Aguayo H.

**Arquitecto de sistemas para entornos extremos**

Diseño infraestructura open-source para condiciones críticas — desde ciudades densas hasta Marte.

## Qué hago

- **Comunicaciones resilientes** — sistemas que mantienen los datos fluyendo incluso cuando fallan satélites o cables submarinos ([Quantum-Flux](https://github.com/enriqueherbertag-lgtm/quantum-flux)).
- **Energía autónoma y soporte vital** — torres que limpian smog urbano ([ShieldAir-Urban](https://github.com/enriqueherbertag-lgtm/ShieldAir-Urban)) y producción de oxígeno en Marte ([ShieldAir-Mars](https://github.com/enriqueherbertag-lgtm/ShieldAir-Mars)).
- **Cuerpos artificiales e interfaces neuronales** — una plataforma física para IA o control humano ([CORPUS](https://github.com/enriqueherbertag-lgtm/Corpus), [ENA](https://github.com/enriqueherbertag-lgtm/ENA)).

## Un ejemplo concreto

**Quantum-Flux** es una capa de comunicaciones tolerante a fallos que divide los datos en múltiples flujos a través de distintos satélites. Aunque fallen 2 o 3 enlaces, el mensaje llega completo. Diseñado para gobiernos, bancos e infraestructura crítica.

## Filosofía

"Reparar el mundo. No reemplazarlo."

Construyo sistemas que trabajan con la infraestructura existente: satélites ya en órbita, edificios ya en ciudades, algoritmos ya probados.
