##  Un ejemplo concreto

**Quantum-Flux** es una capa de comunicaciones tolerante a fallos. Divide los datos en 10 flujos a través de distintos satélites. Aunque fallen 2 o 3 enlaces, el mensaje llega completo.

**Diseñado para:** Gobiernos, bancos e infraestructura crítica que no pueden permitirse perder conectividad.
